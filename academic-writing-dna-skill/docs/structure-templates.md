# 学术论文结构模板 / Academic Paper Structure Templates

> 蒸馏 L2 时参考。4 个子模板覆盖主要论文类型。
> 注意：实际作者可能不完全符合某个子模板，需要标注"实际 X 偏 A，C 偏 B"。

---

## 子模板 A: IMRAD（实验性研究论文）

> Nature / Science / Cell / 领域主流期刊都用这个结构。

### 引言 (Introduction) — 通常 800-1500 字

- **段 1-2**：领域大问题（"X 病每年影响 N 百万人"）
- **段 2-3**：已有研究进展（按时间或主题）
- **段 3-4**：研究缺口 / 未解决问题（"然而，X 仍不清楚...")
- **段 4-5**：本文假设与贡献（"在此，我们报告..." 或 "本文提出..."）

**关键套路**：
- 漏斗结构（宽 → 窄）
- 末段必须有"我们"或"本文"
- 引用密度高（30-50 个引用）

### 方法 (Methods) — 通常 1000-2000 字

按以下顺序：
1. **样本 / 数据来源**（含伦理审批）
2. **实验设计**（含随机化、盲法、对照）
3. **实验流程 / 操作步骤**
4. **数据处理与统计方法**（含软件、显著性阈值）
5. **可重复性**（数据可用性声明、代码仓库）

**关键套路**：
- 时态：过去式
- 详细到他人可复现
- 子节用 "### 2.1, 2.2..." 编号

### 结果 (Results) — 通常 1500-2500 字

按发现分小节，每节围绕 1-2 个图/表：

- **子节命名**：`### X 的表达模式` 而不是 `### 图 1`
- **数据描述先于解释**（"Figure 1a 显示..." 然后 "提示..."）
- **每个图/表必须有 caption**（在文中或末尾图注）

**关键套路**：
- 客观语气，无情绪
- 引用图表：`Figure 1a`, `Table 2`, `(Fig. 1c, arrow)`
- 异常结果也要报告

### 讨论 (Discussion) — 通常 1500-2500 字

5 段经典结构：
1. **核心发现总结**（1 段）
2. **与已有文献比较**（2-3 段：支持 / 不一致 / 解释差异）
3. **机制解释**（1 段：为什么是这样）
4. **局限性**（1 段：明确但简短，不削弱主张）
5. **临床/转化/未来意义**（1-2 段）

**关键套路**：
- 不重复结果细节
- 主动提 1-2 个反方观点并回应
- 末段是"未来方向"或"我们的工作为 X 铺平了道路"

## 子模板 B: 综述 (Review)

> Nature Reviews / Trends in ... / Annual Review of ... 用这个。

### 引言 (Introduction) — 300-500 字

- 综述范围
- 综述的必要性（"鉴于 X 领域的快速进展..."）
- 本综述的组织方式

### 主体 (Body) — 2000-6000 字

按主题 / 时间 / 方法学分类。常见 3 种组织：

**主题分类**（最常见）：
- 每个主题一章
- 主题数量 3-7 个
- 章节按逻辑递进

**时间线**：
- 按历史阶段划分
- 适合回顾性主题

**方法学**：
- 按研究方法分类
- 适合方法导向的综述

### 讨论与展望 (Discussion & Outlook) — 500-1000 字

- 该领域的关键 gaps
- 未来研究方向
- 综述的局限

**关键套路**：
- 用图 / 表 总结关键信息
- 引用密度极高（100-300 个引用）
- 平衡呈现不同观点

## 子模板 C: 学位论文 (Thesis / Dissertation)

> 本科 / 硕士 / 博士论文。章节制，每章可独立。

### 前置部分

- 封面
- 摘要（中英文）
- 目录
- 缩略语表
- 图目录 / 表目录

### 第 1 章：引言 — 3000-5000 字

- 领域背景与重要性
- 文献综述（更详细，比期刊论文长 5-10 倍）
- 本文研究问题与目标
- 论文组织

### 第 2 章：相关工作 / 理论基础 — 3000-5000 字

- 已有方法 / 理论的详细回顾
- 关键概念定义
- 本文的定位

### 第 3-5 章：方法 / 结果 / 讨论（每个研究 1 章）

- 每章自成一体，可对应一篇期刊论文
- 章节结尾常有"小结"段

### 第 6 章：综合讨论 — 3000-5000 字

- 多个发现的整合
- 整体意义
- 局限与未来方向

### 第 7 章：结论 — 1000-2000 字

- 总结主要贡献
- 实践意义
- 未来工作

### 后置部分

- 参考文献
- 致谢
- 附录（数据、代码、补充材料）
- 个人简历 / 在学期间发表论文

**关键套路**：
- 章节制，每章独立
- 文献综述章节特别长
- 比期刊论文多 5-10 倍
- 中英摘要都有

## 子模板 D: 会议论文 (Conference Paper)

> NeurIPS / ICML / ACL / IEEE 等。页数受限（4-8 页）。

### 标题与摘要

- 标题要"卖点前置"（结果/方法关键词）
- 摘要 150-250 字，结构：背景 1 句 + 问题 1 句 + 方法 2-3 句 + 关键结果 2-3 句 + 意义 1 句

### 引言 (Introduction) — 200-400 字

- 比期刊论文短得多
- 1-2 个 motivating example
- 贡献列表（"本文贡献：(1) ..., (2) ..., (3) ..."）

### 相关工作 (Related Work) — 300-500 字

- 子节，按主题分类
- 明确"我们的区别是 X"

### 方法 (Method) — 1000-2000 字

- 算法清晰
- 可有数学公式
- 图表辅助

### 实验 (Experiments) — 800-1500 字

- 数据集描述
- 评估指标
- 主结果（Table 1）
- 消融研究（Ablation）
- 案例分析（可选）

### 结论 (Conclusion) — 100-200 字

- 重述贡献
- 局限（1-2 句）
- 未来工作

### 参考文献 + 附录

- 严格页数限制
- 补充材料可放附录

**关键套路**：
- 卖点前置（"我们达到了 SOTA，比之前最好方法高 X%"）
- 表格密集
- 引用比期刊论文少（15-30 个）

## 如何选择子模板

| 论文特征 | 用哪个 |
| - | - |
| 有 Methods / Results / Discussion 独立章节 | A (IMRAD) |
| 引用密度极高（>50/1000字）且无原始数据 | B (综述) |
| 章节多、有"致谢"和"目录" | C (学位论文) |
| 页数 ≤8、有卖点句、引用少 | D (会议论文) |
| 模糊不清 | A (IMRAD) + 注 "类似 IMRAD 但偏 X" |

## 蒸馏 L2 时怎么用

1. 读完所有论文后，按上表归类每篇
2. 统计每种类型的数量："3 篇 IMRAD + 1 篇综述 + 1 篇会议"
3. 在 Academic-Writing-DNA.md 的 L2 节里：
   - 列出所有 4 个子模板（保留完整结构）
   - 在每个子模板下加 "作者实际用法" 段（具体字数、子节命名、引用密度等）
   - 标注"作者主要用 A"或"作者在 A 和 C 之间切换"
