# Academic-Writing-DNA 输出模板

> 蒸馏完成后，把结果按这个结构写到 `[folder]/Academic-Writing-DNA.md`。
> 总长 ≤ 5000 字。

## L0 学术元信息

- **目标作者/期刊**: [name]
- **论文类型**: [IMRAD | 综述 | 学位论文 | 会议 | 混合]
- **语言**: [en | zh | mixed]
- **字数范围**: [N-M 字]
- **参考文献格式**: [Vancouver | APA | AMA | ...]
- **投稿约束**: [字数上限 / 图表数 / 摘要字数 / ...]

## L1 学术语言DNA

- **平均句长**: X 字
- **短句 (≤15字) 占比**: X%
- **长句 (≥50字) 占比**: X%
- **被动语态比例**: X%
- **人称偏好**: [第一人称复数 "我们" / 无人称 / ...]
- **高频术语 Top 10**: [term1 (N 次), term2, ...]

- **标点偏好**: [破折号 / 冒号 / 分号 的密度]

## L2 学术结构模板

### 子模板 A: IMRAD (实验性)
- 引言: 背景 → 研究 gap → 本文贡献
- 方法: 子节列表
- 结果: 配图表描述
- 讨论: 主张 → 证据 → 限定 → 反驳 → 收束

### 子模板 B: 综述
- 引言: 范围与综述必要性
- 主题分类: 2-5 个并列主题
- 讨论与展望

### 子模板 C: 学位论文
- 摘要 / 引言 / 文献综述 / 方法 / 结果 / 讨论 / 结论 / 致谢

### 子模板 D: 会议论文
- 卖点前置摘要 → 引言(短) → 方法 → 结果 → 结论

(实际写作时根据目标论文类型选对应子模板，删去其他)

## L3 选题逻辑

- **写作时机**: [早期判断 / 跟进分析 / 事后复盘]

- **不写的话题**: [1-3 个]
- **研究 gap 识别方式**: [从最新文献缺口 / 从临床未解决问题 / ...]

## L4 素材策略

- **引文密度**: X / 1000 字

- **证据等级偏好**: [一手数据 > 二手综述 > 案例 > 截图]
- **数据呈现**: [统计显著性 / 效应量 / 可信区间 / ...]
- **自引率**: X%

## L5 认知框架

- **核心命题 (3-5 条)**: [proposition 1, 2, 3]
- **论证模式**: [主张 → 证据 → 限定 → 反驳 → 收束]
- **立场-反方-回应**: [怎么处理反对意见]
- **价值判断倾向**: [进步主义 / 保守 / 平衡 / ...]

## L6 视觉风格

- **平均图表数**: X
- **平均表格数**: X
- **图题平均字数**: X
- **图表类型**: [数据图 / 示意图 / 流程图 / 照片]
- **图文协作模式**: [图即论证 / 引导语→图→解读 / 文字概括+图精确]
- **排版密度**: [段落平均 X 字 / 是否有刻意的短段落节奏]

## 使用说明

### 适用场景
- 写 [paper_type] 论文时，遵循 L0-L6 规则
- 套话必须规避（L1 列出的）
- 结构按 L2 对应子模板
- 论证按 L5 的模式
- 图表数量和类型参考 L6

### 不适用
- 跨学科写作（除非 L3 明示支持）
- 短篇科普文
- 与原风格差距很大的新话题

## 边界

本 DNA 仅用于学习与个人写作资产沉淀。禁止：
- 冒充原作者
- 抄袭
- 绕过查重

## 写作时如何使用这个文件

当用户说"帮我写..."时，Claude：
1. 找到这个 DNA 文件
2. 问用户"要用这个风格吗？"
3. 用户同意后，按 L0-L6 规则写
4. 不再问论文类型/期刊/字数（DNA 已有）

## 写多少字合适

- 蒸馏了 1 篇：演示模式，DNA 可能不稳定
- 蒸馏了 3-5 篇：可用
- 蒸馏了 5-10 篇：充分
- 蒸馏了 >10 篇：最稳
